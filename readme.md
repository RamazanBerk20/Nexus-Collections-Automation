## Usage

To start the application, use the following command in the command prompt:

```sh
main.py {process name of your browser}.exe
```

For Microsoft Edge, the command would be:

```sh
main.py msedge.exe
```

For Linux however, it would be like:
```sh
main.py firefox
```
